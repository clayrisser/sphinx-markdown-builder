## Installation

```sh
pip3 install sphinx-markdown-builder
```


## Dependencies

* [Python 3](https://www.python.org)


## Usage

Build markdown files with Makefile

```sh
make markdown
```

Build markdown files with `sphinx-build` command

```sh
cd docs
sphinx-build -M markdown ./ build
```
